💠 SEMANA 03 — Bootstrap y Tailwind CSS

📘 Describir temas aprendidos con sus propias palabras

Durante esta semana se estudiaron los frameworks de diseño Bootstrap y Tailwind CSS. Bootstrap proporciona un sistema de grillas y componentes predefinidos (botones, tarjetas, menús), facilitando el desarrollo rápido de interfaces. Tailwind, en cambio, adopta un enfoque utility-first, donde los estilos se aplican mediante clases directas. Se comprendió cómo ambos frameworks simplifican el desarrollo frontend, favoreciendo el diseño responsivo y la personalización.

🧪 Ejercicios de laboratorio – Resultados

Se construyó una página con el sistema de grillas de Bootstrap, aplicando componentes y modificando su diseño con variables CSS. Posteriormente, se replicó el diseño usando Tailwind CSS, controlando espaciados, colores y tipografía con clases utilitarias. El resultado fue un sitio adaptable a distintos dispositivos, optimizado y visualmente moderno.

💭 Reflexión: ¿Qué aprendió? y ¿Cómo aprendió?

Aprendí a emplear frameworks para agilizar el diseño de interfaces responsivas. Al comparar Bootstrap y Tailwind, comprendí las ventajas de cada uno según el contexto: rapidez con Bootstrap y personalización con Tailwind. Aprendí practicando y experimentando directamente con ambos en proyectos reales, fortaleciendo mi dominio en diseño adaptativo.
