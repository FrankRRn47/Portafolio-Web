⚙️ SEMANA 04 — JavaScript y TypeScript

📘 Describir temas aprendidos con sus propias palabras

Se profundizó en JavaScript, comprendiendo sus tipos de datos (numéricos, cadenas, booleanos, objetos), operadores, funciones y estructuras de control. También se aplicaron diferentes formas de declarar funciones (normales, flecha, anónimas, closures). Además, se introdujo TypeScript, que añade tipado estático y detección temprana de errores, mejorando la robustez del código. Finalmente, se exploró el uso del elemento <canvas> para crear animaciones y gráficos mediante coordenadas.

🧪 Ejercicios de laboratorio – Resultados

En el Laboratorio 04, se desarrolló una animación utilizando el elemento <canvas> y funciones JavaScript. Se controlaron colores, movimiento y eventos del mouse. El resultado fue una animación interactiva en la que los elementos respondían al movimiento del cursor en tiempo real.

💭 Reflexión: ¿Qué aprendió? y ¿Cómo aprendió?

Aprendí a incorporar lógica e interactividad en las páginas web mediante JavaScript y a estructurar código más seguro con TypeScript. Comprendí la importancia de controlar el DOM y manejar eventos dinámicos. Aprendí practicando con ejemplos funcionales y analizando los resultados visuales obtenidos en el navegador.
