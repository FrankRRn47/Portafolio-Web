🧩 SEMANA 01 — Fundamentos de Tecnologías Web

📘 Describir temas aprendidos con sus propias palabras

Durante esta semana se abordaron los principios básicos de las tecnologías web. Se comprendió la diferencia entre los conceptos de sistema web, aplicación web, sitio web y página web. Un sistema web integra múltiples aplicaciones conectadas entre sí; una aplicación web es un software que se ejecuta en el navegador; un sitio web está formado por un conjunto de páginas relacionadas bajo un mismo dominio; y una página web representa un documento individual dentro del sitio.

También se analizaron las tecnologías esenciales: HTML define la estructura, CSS el diseño visual y JavaScript la interactividad. Se revisó cómo funciona Internet, incluyendo el papel del DNS (traducción de nombres a IP), TCP/IP (transmisión de datos) y HTTP (protocolo de comunicación entre cliente y servidor). Además, se exploraron los roles en el desarrollo web: el frontend se ocupa de la interfaz, el backend de la lógica y base de datos, y el fullstack combina ambos. Finalmente, se instaló y configuró Visual Studio Code, aprendiendo sus herramientas, extensiones y atajos.

🧪 Ejercicios de laboratorio – Resultados

Se realizó el Laboratorio 01A, que consistió en familiarizarse con la interfaz de Visual Studio Code, instalando extensiones y practicando comandos. También se practicó el uso de Emmet, herramienta que permite generar estructuras HTML rápidamente. Por ejemplo, con la abreviatura: 

div.container>header#header{header}+nav#nav{nav}+section#section{section>article#article{article}^}+aside#aside{aside}+footer#footer{footer}

Se obtuvo una estructura HTML semántica completa en segundos, demostrando la eficiencia del uso de abreviaturas para acelerar la codificación.

💭 Reflexión: ¿Qué aprendió? y ¿Cómo aprendió?

Aprendí los fundamentos de cómo funciona la web y las diferencias entre sus principales componentes. Comprendí el papel de cada rol dentro del desarrollo y la utilidad de Visual Studio Code para escribir código de forma eficiente. Mediante la práctica con Emmet, desarrollé rapidez y precisión al generar estructuras HTML, entendiendo que la automatización acelera significativamente el flujo de trabajo.
