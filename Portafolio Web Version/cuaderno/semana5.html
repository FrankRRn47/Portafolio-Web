💠 SEMANA 05 — Framework JS y Componentes

📘 Describir temas aprendidos con sus propias palabras

Esta semana se estudió el Framework React, comprendiendo su funcionamiento basado en componentes y su mecanismo de client-side rendering. Se aprendió sobre JSX, props, children y la arquitectura modular del frontend moderno. También se practicó el uso de herramientas de desarrollo como npm y Vite para gestionar dependencias y crear proyectos. Además, se abordaron diferentes técnicas para aplicar estilos, desde hojas externas hasta styled components.

🧪 Ejercicios de laboratorio – Resultados

El Laboratorio 05A consistió en crear componentes funcionales en React, pasando datos del componente padre al hijo mediante props y mostrándolos dinámicamente en pantalla. Se utilizó Vite para el entorno de desarrollo y se aplicaron estilos modulares para mantener el código organizado. El resultado fue una interfaz interactiva con componentes reutilizables y comunicación fluida entre ellos.

💡 Reflexión: ¿Qué aprendió? y ¿Cómo aprendió?

Aprendí el enfoque moderno del desarrollo web basado en componentes reutilizables, comprendiendo cómo React organiza la interfaz y actualiza el DOM de manera eficiente. Aprendí practicando la estructura y comunicación entre componentes, reforzando la lógica de modularidad, mantenimiento y escalabilidad en el código. Este aprendizaje fortaleció mi comprensión del desarrollo frontend contemporáneo.
