Semana 6 — Eventos, APIs y Routing

🔗 SEMANA 06 — Eventos, APIs y Routing

📘 Describir temas aprendidos con sus propias palabras

Durante esta semana se profundizó en el manejo de eventos, renderizado condicional e iterativo, así como en la creación de formularios interactivos. También se aprendió a implementar rutas para la navegación entre vistas y a consumir APIs externas mediante promesas y async/await, utilizando la librería Axios para obtener y mostrar datos dinámicos. Se comprendió cómo estos elementos permiten desarrollar aplicaciones web más dinámicas, conectadas y escalables.

🧪 Ejercicios de laboratorio – Resultados

En el Laboratorio 06A, se desarrolló una aplicación React que muestra datos obtenidos de una API pública. Se manejaron eventos para actualizar la interfaz según las interacciones del usuario y se aplicó renderizado condicional para controlar la visibilidad de los componentes. El resultado fue una interfaz funcional y dinámica que consumía información real desde un servicio externo, demostrando la integración efectiva del frontend con el backend.

💡 Reflexión: ¿Qué aprendió? y ¿Cómo aprendió?

Aprendí cómo conectar una aplicación web con servicios externos, comprendiendo el flujo asíncrono de datos y la importancia de manejar correctamente las promesas y los errores. También entendí cómo los eventos permiten crear interfaces más interactivas y personalizadas. Aprendí mediante la experimentación directa con APIs, lo que me permitió visualizar de forma práctica la integración entre el frontend y el backend, fortaleciendo mis habilidades en desarrollo de aplicaciones dinámicas.
