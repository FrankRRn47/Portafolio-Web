Semana 02 — HTML, XML y CSS

🌐 SEMANA 02 — HTML, XML y CSS

📘 Describir temas aprendidos con sus propias palabras

En esta semana se profundizó en la estructura de las páginas web con HTML5 y las diferencias con XML, que es un lenguaje más estricto usado para el intercambio de datos. Se comprendió la jerarquía del árbol DOM (Document Object Model) y el proceso de renderizado que realiza el navegador.

También se introdujeron conceptos de SEO (optimización para motores de búsqueda) y compatibilidad entre navegadores, destacando la importancia de etiquetas semánticas y atributos adecuados. En cuanto a CSS3, se aprendió la sintaxis básica, los tipos de display (block, inline, inline-block), las unidades absolutas y relativas, y las técnicas de Flexbox y Grid Layout para lograr diseños adaptables y responsivos.

🧪 Ejercicios de laboratorio – Resultados

Se desarrolló una página web básica aplicando etiquetas semánticas, imágenes, listas y enlaces. Luego se añadió CSS para definir el diseño responsivo, empleando propiedades como display, margin, padding y grid-template-columns. El resultado fue una página correctamente estructurada y visualmente equilibrada, con un diseño adaptable a diferentes dispositivos.

💭 Reflexión: ¿Qué aprendió? y ¿Cómo aprendió?

Aprendí a construir la base de una página web limpia y ordenada, comprendiendo el valor del HTML semántico y del CSS para definir la apariencia y adaptación visual. A través de la práctica en laboratorio, logré entender cómo el navegador interpreta el código y cómo aplicar buenas prácticas para optimizar el rendimiento y la accesibilidad. Este aprendizaje permitió fortalecer las bases para el desarrollo web moderno.
