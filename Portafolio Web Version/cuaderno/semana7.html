⚡ SEMANA 07 — Hooks en Framework JS

📘 Describir temas aprendidos con sus propias palabras

Esta semana se exploraron los Hooks de React, herramientas fundamentales que permiten manejar el estado y el ciclo de vida de los componentes funcionales. Se practicó con useState para gestionar variables reactivas, useEffect para controlar efectos secundarios como llamadas a APIs o eventos del DOM, y useContext para compartir datos globales sin necesidad de props. Además, se estudiaron hooks avanzados como useRef, useReducer, useCallback y useMemo, los cuales optimizan el rendimiento y la reutilización del código. Finalmente, se aprendió a crear hooks personalizados para encapsular y compartir lógica entre componentes, mejorando la modularidad del proyecto.

🧪 Ejercicios de laboratorio – Resultados

En el laboratorio correspondiente, se desarrolló una aplicación que implementa varios hooks para gestionar estados, efectos secundarios y referencias. Se creó un hook personalizado que controlaba los valores y validaciones de un formulario, logrando una estructura más limpia y escalable. Asimismo, se aplicaron useEffect para actualizar datos dinámicamente y useCallback para optimizar funciones dependientes de estado. El resultado fue una aplicación más organizada, eficiente y con mejor rendimiento.

💡 Reflexión: ¿Qué aprendió? y ¿Cómo aprendió?

Aprendí a manejar de forma avanzada el estado interno y la lógica funcional de los componentes React utilizando hooks, comprendiendo su papel en la composición y reutilización de lógica. También entendí cómo optimizar el rendimiento evitando renderizados innecesarios. Aprendí mediante la experimentación práctica con ejemplos de uso real, lo que permitió reforzar la comprensión de cada hook y su aplicabilidad dentro de proyectos complejos.
